# 🌍 Go Burak Go — Dijital İçerik Asistanı

13 ülkeli dünya turuna çıkan Burak Yıldız'ın viral içerik üretim platformu.

![Go Burak Go](public/images/hero-bg.jpg)

## ✨ Özellikler

- 🎬 **Senaryo Üretimi** — Hook → Story → CTA formatında viral içerikler
- 📈 **Trend Analizi** — Reddit, TikTok, YouTube trendleri
- 📅 **Yayın Takvimi** — Otomatik paylaşım planlaması
- 🗺️ **Dünya Turu** — 13 ülke rotası takibi
- 📱 **PWA** — Mobil uygulama olarak kullanılabilir
- 🚀 **Lansman Öncesi** — Stok videolarla hazır içerikler

## 🛠️ Teknolojiler

- **Next.js 16** — React framework
- **PostgreSQL** — Veritabanı
- **Drizzle ORM** — Database toolkit
- **Tailwind CSS** — Styling
- **PWA** — Progressive Web App

## 🚀 Kurulum

### Vercel'e Deploy (Önerilen)

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/brkyldz71/go-burak-go)

### Lokal Geliştirme

```bash
# Repoyu klonla
git clone https://github.com/brkyldz71/go-burak-go.git
cd go-burak-go

# Bağımlılıkları yükle
npm install

# .env dosyası oluştur
cp .env.example .env

# Veritabanını hazırla
npx drizzle-kit push

# Geliştirme sunucusunu başlat
npm run dev
```

## 📱 PWA Kurulumu

### iPhone
1. Safari'de siteyi aç
2. Paylaş butonuna (⬆️) bas
3. "Ana Ekrana Ekle" seç

### Android
1. Chrome'da siteyi aç
2. "Ana ekrana ekle" banner'ına tıkla

### Masaüstü
1. Chrome/Edge'de siteyi aç
2. Adres çubuğundaki 📥 ikonuna tıkla
